# Deep Research Agent

The Deep Research Agent is a graph-based AI assistant that replaces hours of manual research with a single query. It pulls live data from Google, Bing, and Reddit simultaneously via [Bright Data](https://brightdata.com), analyzes each source independently with GPT-4, and delivers a synthesized answer with full citations — all in about a minute.

## Why Deep Research Agent?

Most LLM assistants rely on a single API or a cached knowledge corpus. This leads to incomplete, biased, or outdated answers — especially for competitive analysis, market research, or real-world decision-making.

The Deep Research Agent solves this by:

- Querying **multiple live sources** in parallel (Google SERPs, Bing SERPs, Reddit threads)
- Reliably scraping and normalizing results including Reddit posts and comments
- Applying **layered LLM analysis** — per-source summaries, then cross-source synthesis
- Enforcing **typed outputs with citations** for every claim, making answers auditable

## How it works

```
User Query
    │
    ▼
LangGraph Controller (State)
    ├─ Parallel: google_search, bing_search, reddit_search
    ├─ analyze_reddit_posts  →  retrieve_reddit_posts
    ├─ Parallel: analyze_google_results, analyze_bing_results, analyze_reddit_results
    └─ synthesize_analyses  →  Final Answer (with citations)
```

The agent runs three live searches in parallel, then retrieves full Reddit post data for the most relevant threads. Independent per-source analysis nodes run in parallel before a final synthesis node merges everything into a structured, cited response.
